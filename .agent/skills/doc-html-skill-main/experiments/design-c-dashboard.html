☰Summary

  • rf-mcp is the handshake between your AI agent and Robot Framework. Plain words in, a working .robot suite out.
  • It does not guess — keywords run in a live RF context, step by step, with real feedback before a single line is written down.
  • Three execution backends, because one size never fits all.
  • Built on FastMCP and RF 7, with small-LLM optimizations so lighter models keep up.

→From plain words to a passing suite

1Describe intent

2Discover keywords

3Execute stepwise
